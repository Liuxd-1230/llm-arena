# LLM Arena - 大模型能力盲测评估平台 v3

> 10个评估维度 × 4个难度等级 × 双盲测试 × 自动评分 × 48道系统化题目

## 核心特性

### 双盲测试
- **收集阶段**：管理员收集多个模型的回答，系统分配匿名盲测ID（#001, #002...）
- **盲测阶段**：全屏展示渲染结果，评分者看不到模型名称和代码风格
- **评分阶段**：逐个打分，支持手动评分和自动评分
- **揭盲阶段**：所有评分完成后，一键揭示盲测ID与模型名的对应关系

### 自动评分（客观题）
以下维度支持自动评分，无需人工介入：
- **算法与编程**：检测算法模式、时间复杂度、代码结构
- **逻辑推理**：验证最终答案正确性和推理过程
- **指令遵循**：格式合规性检查（JSON结构、字数约束等）

### 手动评分（主观题）
以下维度需要人工盲测评分：
- **前端代码美感**：缩略图盲测 + 全屏预览 + 悬浮打分面板
- **写作能力**：全屏文本展示
- **创意思维**：全屏文本展示
- **中文能力**：全屏文本展示
- **安全伦理**：全屏文本展示
- **知识问答**：全屏文本展示
- **长文本理解**：全屏文本展示

## 前端题目（12道）

前端评测采用缩略图盲测模式，支持4套动态评分Profile：

| Profile | 适用题目 | 核心权重 |
|---------|---------|---------|
| `ui_general` | 青铜/白银通用 | 视觉30 / 布局25 / 交互20 / 响应式15 / 代码10 |
| `ui_design_system` | 设计系统类 | 系统一致性30 / CSS架构25 / 组件状态20 / 文档层级15 / 主题切换10 |
| `ui_dashboard_interactive` | 仪表盘/交互类 | 信息架构25 / 交互可用性25 / 图表表达20 / 响应式15 / 代码管理15 |
| `ui_creative` | 创意视觉类 | 视觉冲击30 / 创意完成度25 / 动效节奏20 / 布局可读性15 / 技术约束10 |

## 使用流程

```
1. 选择评估维度
2. 选择难度和题目
3. 复制Prompt发给被测模型
4. 粘贴模型回答 → 添加到待测列表
5. 重复步骤2-4，收集所有模型的答案
6. 点击"开始盲测"
   → 前端代码：缩略图盲测 + 全屏预览
   → 客观题：自动评分
   → 主观题：全屏文本展示
7. 逐个打分
8. 查看排行榜
9. 揭盲 → 看到模型名对应关系
```

## 本地运行

**⚠️ 必须使用HTTP服务器，不能直接双击打开（ES模块需要HTTP协议）**

```bash
# 方式1：Python
python -m http.server 8000

# 方式2：Node.js
npx serve .

# 方式3：VS Code
# 安装 Live Server 插件，右键 index.html → Open with Live Server
```

然后打开 http://localhost:8000（或对应端口）
