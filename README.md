# LLM Arena - 大模型能力盲测评估平台

> 10个评估维度 × 4个难度等级 × 双盲测试 × 自动评分 × 功能检测

## 核心特性

### 双盲测试
- **收集阶段**：管理员收集多个模型的回答，系统分配匿名盲测ID（#001, #002...）
- **盲测阶段**：全屏展示渲染结果，评分者看不到模型名称和代码风格
- **评分阶段**：逐个打分，支持手动评分和自动评分
- **揭盲阶段**：所有评分完成后，一键揭示盲测ID与模型名的对应关系

### 自动评分（客观题）
以下维度支持自动评分，无需人工介入：
- **算法与编程**：检测算法模式、时间复杂度、代码结构
- **逻辑推理**：验证最终答案正确性和推理过程
- **知识问答**：关键词匹配检查知识点覆盖度
- **指令遵循**：格式合规性检查（JSON结构、字数约束等）
- **长文本理解**：信息提取准确性验证

### 手动评分（主观题）
以下维度需要人工盲测评分：
- **前端代码美感**：全屏iframe渲染，不显示代码
- **写作能力**：全屏文本展示
- **创意思维**：全屏文本展示
- **中文能力**：全屏文本展示
- **安全伦理**：全屏文本展示

### 前端代码功能检测
提交前端代码前，自动检测10项功能指标：
代码可渲染 / DOCTYPE / CSS样式 / 悬停状态 / 过渡动画 / 响应式 / 语义化HTML / 交互元素

## 使用流程

```
1. 选择评估维度
2. 选择难度和题目
3. 复制Prompt发给被测模型
4. 粘贴模型回答 → 添加到待测列表
5. 重复步骤2-4，收集所有模型的答案
6. 点击"开始盲测"
   → 前端代码：全屏渲染预览
   → 客观题：自动评分
   → 主观题：全屏文本展示
7. 逐个打分
8. 查看排行榜
9. 揭盲 → 看到模型名对应关系
```

## 技术栈

- 纯前端实现，单HTML文件，无需后端
- Geist字体（Vercel设计系统）
- RemixIcon图标库
- localStorage持久化
- GitHub Pages可直接部署

## 本地运行

```bash
# 直接用浏览器打开
open index.html

# 或启动本地服务器
python3 -m http.server 8080
```
